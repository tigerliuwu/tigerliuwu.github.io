# conda help
```
$ conda help
$ conda -h
$ conda --help
```

显示conda支持的所有的命令。

# conda search
```
conda search panda
```

这个命令会去anaconda的repositories去搜索所有相关的modules.<https://repo.continuum.io/pkgs/>
